## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/LuizGustavoSena/Pokedex.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Pokedex
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

---

## Usage

### Run the Development Server
```bash
npm start
```

### Build for Production
```bash
npm run build
```

### Run Tests
```bash
npm test
```
